# LookbookMenu Website

Marketing website for LookbookMenu — a photo-first digital menu product for restaurants.

## Quick Start

```bash
# Landing page (no build needed)
open index.html

# Try-It page (Vue app, needs Vite)
npm install
npm run dev
```

## Stack

- **Landing page:** Vanilla HTML, CSS, JS (no frameworks)
- **Try-It page:** Vue 3 + Vue Router, built with Vite
- **Hosting:** Cloudflare Pages (auto-deploys from `main`)
- **Media:** Cloudflare R2
